Algorithm for Finding Modular Inverse

Necessary Definitions

Modular Inverse
For a natural number \(n\) and an integer \(a\), we say that \(b\) is an inverse of \(a\) \(\pmod n\) if \[ ab \equiv 1 \pmod{n}. \]

Designing Algorithm for finding the mod inverse

So the problem for this algorithm will be the following.

Input: An integer \(a\) and a natual number \(n\) such that \(\gcd(a,n)=1\)
Output: A natural number \(b\) such that \(ab \equiv 1\pmod{n}\)

A very bad Algorithm


Brute Force Algorithm:
1. Initialize i = 1;
2. For i from 1 to n-1:
3.   check a*b (mod n);
4.      if a*b (mod n) == 1{
            return b; }
5.   End if
6. End for

This code is bad as the running time of it is exponential.
